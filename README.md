## 📖 Description

DevPath Junior est une plateforme interactive conçue pour aider les développeurs juniors à se préparer efficacement aux entretiens techniques. Le site propose plusieurs outils et jeux pour s'entraîner de manière ludique et structurée.

---

## ✨ Fonctionnalités

### 📝 To-Do List des étapes
Liste interactive pour suivre sa progression dans la préparation aux entretiens.

### 📄 Générateur de README
Outil pour créer rapidement des README professionnels pour son profil GitHub.

### 🎮 Jeux d'entraînement
- **Speed Challenge QCM** : Questions techniques en mode chrono pour tester ses connaissances
- **Memory Cards** : Jeu de cartes pour mémoriser les définitions techniques

### 💼 Simulateur d'entretien
Simulation d'entretien avec des conseils de réponses adaptés aux questions courantes.

---

## 🛠️ Technologies utilisées

- **Frontend** : HTML5, CSS3, JavaScript (Vanilla)
- **Data** : JSON pour le stockage des questions et définitions
- **Design** : Maquette Figma + Cahier des charges complet
- **Responsive** : Compatible mobile, tablette et desktop

---

## 🎨 Processus de développement

1. **Conception** : Création de la maquette Figma
2. **Planification** : Rédaction du cahier des charges
3. **Développement** : Implémentation des fonctionnalités
4. **Responsive Design** : Adaptation pour tous les écrans
